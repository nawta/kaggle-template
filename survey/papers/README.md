# 論文調査ガイド

## 概要

コンペに関連する論文の調査結果をこのディレクトリに蓄積する。

## ワークフロー

### 1. 論文の検索

- **Google Scholar**: キーワード検索、被引用数でのフィルタリング
- **Semantic Scholar**: API を活用した効率的な検索
- **arXiv**: 最新のプレプリント検索
- **過去コンペの上位解法**: Kaggle Discussion から参照される論文

### 2. 論文のまとめ方

各論文について以下の形式でまとめる：

```markdown
# {論文タイトル}

- **著者**:
- **年**:
- **URL**:
- **キーワード**:

## 概要
（1-2文で論文の主張を要約）

## 手法の詳細
（コンペに関連する手法の説明）

## コンペへの適用可能性
（この論文の知見をどう活かせるか）

## 実装メモ
（実装時の注意点、参考コード等）
```

### 3. ファイル命名規則

`{YYYY}_{著者名}_{短縮タイトル}.md`

例: `2023_smith_efficient_attention.md`

## ディレクトリ構成

```
papers/
├── README.md          # このファイル
├── .gitkeep
└── *.md               # 各論文のまとめ
```
